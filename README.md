# Quote Editor App 

This is me trying to do a the quote-editor example.

## Getting Started
- Run `git clone git@github.com:fercreek/quote-editor.git`
- Go to the folder
- `npm install`
- `bundle install`
- `rails db:create`
- `rails db:migrate`

## Run application
- `bin/dev`

## Run tests
- `bin/rails db:seed`
- `bin/rails test:all`
